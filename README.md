# HISTORIZANDO
  Site com perguntas sobre historia, através de quiz interativo e correção automática das questões!
  
  Este site foi feito com:
  * HTML
  * CSS 
  * JavaScript
  
  Contém:
  * Questões de história
  * Alternativas dinamicas
  * Script que dar a resposta da questão automaticamente após marcada
  * Sessão com comentários sobre a questão
